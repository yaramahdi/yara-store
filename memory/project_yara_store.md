Built a complete static e-commerce website for "متجر يارا" (Yara Store).

**Files:** index.html, style.css, app.js, admin.html (all in c:\Users\yaram\Downloads\yara store\)

**Why:** User wants a mobile-first Arabic RTL online store to sell women's clothing (abayas, dresses, blouses, sets).

**Key features:**
- Product grid with real photos, original price (strikethrough) + sale price + discount %
- Product detail modal with size selection and quantity
- Cart sidebar → 3-step checkout: customer form → invoice → payment
- Payment: Islamic Bank account + Bank of Palestine wallet (editable from admin)
- WhatsApp confirmation button sends formatted order to owner's number
- Admin panel (admin.html) with password "yara2024" — add/edit/delete products, upload images, set sizes, prices, bank details, WhatsApp number
- All data stored in localStorage

**How to apply:** If user wants to add features or fix issues, all 4 files are in the yara store folder. The admin panel at admin.html is the database for products.
